# Fake-Url-Dictator
Fake URL Detector is a cybersecurity project developed using Java and MySQL to identify suspicious or phishing website links. The main purpose of this project is to help users stay safe from online scams, fake websites, and data theft. Many attackers create URLs that look similar to trusted websites, and users may click them unknowingly. This system helps detect such links before they are opened.

The application allows users to enter any website URL and checks it using stored database records and predefined detection rules. It analyzes common phishing patterns such as unusual domain names, extra symbols, misspelled brand names, shortened links, and insecure formats. After scanning, the system displays whether the URL is safe, suspicious, or fake.

The project uses Java for frontend interface and backend logic, MySQL for storing blacklist and whitelist URLs, and JDBC for connecting the application with the database. It is simple, fast, and user-friendly, making it suitable for students and beginners learning cybersecurity concepts.

This project can be improved further by adding machine learning features, real-time online URL scanning, and browser extension support. Fake URL Detector is a practical project that spreads awareness about internet security and safe browsing.
